# Analytics

The `analytics` template provides deep insights and performance reviews.

## Pages in this directory
- `dashboard.html`: The main analytical chart hub for reviewing past performance and financial impact.

## Functionalities
- **Forecast Accuracy Tracking**: Tracks the MAPE (Mean Absolute Percentage Error) to show you how accurate the predictions have been.
- **Financial Impact**: Summarizes the monetary value of profits gained and losses saved by following system recommendations.
- **Marketplace Performance**: Granular comparison of revenue, units sold, and profit across different sales channels.
- **Revenue trend Chart**: Visualizes daily revenue trends.
- **Exporting Options**: Export analytical metrics to CSV.

## How to Use
1. Visit the Analytics page from the sidebar.
2. Use the date toggles (7d, 14d, 30d, 90d) to adjust the timeframe of data being analyzed.
3. Review the KPIs at the top and investigate any marketplace that might be underperforming in the detailed tables and charts below.

## Do's and Don'ts
*   **Do use analytics to determine marketplace priority.** If one platform consistently drives the highest portion of revenue, consider increasing its priority in the Marketplaces setup.
*   **Do track the "Recommendation Follow Rate."** If it's low, ensure your team is using the allocation planner and restocking when it tells you to.
*   **Don't assume forecasts are always perfect.** Check the "Forecast Accuracy" metric; if accuracy drops, be slightly more conservative with automated purchasing or dispatching.
