# Authentication

The `auth` templates manage the user lifecycle (login, registration).

## Pages in this directory
- `login.html`: Entry point for registered users.
- `register.html`: Onboards new users and creates default marketplace settings.

## Functionalities
- Secure login and registration.
- Automatically seeds initial standard marketplaces (Amazon, Flipkart, Meesho) for new accounts.

## How to Use
- Navigate to the root URL `/` to be directed to the login page. Use credentials or create a new account via the register screen.

## Do's and Don'ts
*   **Do use strong passwords** as your business data and sales metrics will be stored here.
*   **Don't share accounts.** Since allocations and sales are tied to the active user's environment, keep logins secure and individual.
